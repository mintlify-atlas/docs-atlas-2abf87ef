## Overview

Development and design of web solutions for various professionals and businesses. These projects primarily included two nutrition professionals and an English learning services company.

</Note>

## Project 1: Nutritionist - Lic. Cyntia Concetti

<Card title="Visit Website" icon="link" href="https://nutricioncynconcetti.com.ar/">
  View the live website at nutricioncynconcetti.com.ar
</Card>

### Project Requirements

In this case, a page was requested where some of her services could be made known, social networks displayed, and a contact form included.

### Implementation

<AccordionGroup>
  <Accordion title="Technology Stack">
    The page was developed with HTML, CSS, JavaScript, PHP, and MySQL.
  </Accordion>
  
  <Accordion title="Contact Form">
    Implemented a contact form using PHP's mail() function to handle inquiries.
  </Accordion>
  
  <Accordion title="Database System">
    Database system to store data from users who register on the page.
  </Accordion>
  
  <Accordion title="Admin Login System">
    Login system so the client can access the database and view registered user data.
  </Accordion>
  
  <Accordion title="Responsive Design">
    Developed with responsive design to display correctly on any device.
  </Accordion>
</AccordionGroup>

<Card title="Visit Website" icon="link" href="https://speakit.com.ar/">
  View the live website at speakit.com.ar
</Card>

### Project Requirements

This was my first project for a group of professionals, including my sister, and it was a much more complex project due to having implemented everything from scratch without the help of any framework, which gave me deep knowledge in how web development works.

<Warning>
  **Status:** This project is still in development
</Warning>

### Implementation

<AccordionGroup>
  <Accordion title="Core Features">
    The page needed to have:
    - Multiple information sections about the services provided
    - Contact form
    - Responsive design for correct display on any device
  </Accordion>
  
  <Accordion title="Payment System (In Development)">
    Soon they will also be able to offer their services on the page. To pay for them online, a payment system was implemented with Mercado Pago, allowing users to pay with credit card, debit card, bank transfer, PagoFacil, Rapipago, etc.
