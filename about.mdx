### Responsive Layout Implementation

The About page uses a sophisticated responsive grid system:

<Tabs>
  <Tab title="Mobile (< 50em)">
    - Single column layout
    - Vertical stack with 3.5rem gap
    - Full-width content sections
    - Touch-optimized spacing
  </Tab>
  
  <Tab title="Desktop (≥ 50em)">
    - Three-column grid: `1fr 60% 1fr`
    - Content centered in middle column
    - Section titles in first column
    - Section content spans columns 2-4
    - Enhanced typography sizing
  </Tab>
</Tabs>
